# Contributing to Carbon Shadow

## Project Structure
- `index.html`, `dashboard.html`, `camera.html` - Core pages
- `*.js` - Logic files
- `*.css` - Styling files
- `tests/` - Automated tests using Vitest

## How to run locally
Run the following commands:
```sh
npm install
npm run dev
```

## How to execute tests
Run the following command:
```sh
npm test
```

## How to deploy
The project is statically hosted on Vercel. Pushing to the `main` branch automatically triggers a deployment.

## Coding conventions
- Pure HTML/CSS/JS without compilation steps
- Meaningful variable names
- Code formatting with 2 spaces indentation
